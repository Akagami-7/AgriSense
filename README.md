# AgriSense (V2.1): Earthy & Accurate 🌿

AgriSense is a premium, AI-powered Smart Farming Assistant designed for the modern agronomist. This update (V2.1) focuses on a "Slightly Earthy" professional aesthetic and high-precision data accuracy.

## 🌟 V2.1 Highlights

### 🏝️ Earthy Island Navigation
We've replaced the interactive sidebar with a **Static Earthy Island** top-bar. It features a nature-inspired palette (`#fdfaf6` / `#1a1614`) and permanent navigation labels for maximum usability and a premium, grounded feel.

### 🎯 High-Precision AI Scaling
All AI diagnostic outputs are now scaled to enterprise standards. Confidence scores (e.g., `0.95`) are automatically converted to human-readable percentages (**95%**) across both Crop and Soil intelligence modules.

### 📊 Synchronized Field Dashboard
The dashboard trends and health distribution charts are now perfectly synchronized with your real-time session scan history. Whether you have 2 scans or 200, the visualization scales to provide accurate, per-scan granularity.

---

## 🎨 Design Philosophy
AgriSense follows a **Nature-Inspired Design System**, blending professional SaaS aesthetics with earthy tones and vibrant agricultural palettes. Key design features include:
- **Premium Earthy Palette**: Hand-curated colors inspired by soil, sage, and forest canopy.
- **Glassmorphism & Soft Gradients**: Creating a modern, state-of-the-art interface.
- **Dynamic Session Security**: All data (history and API keys) is purged upon browser closure, ensuring your field data stays private.
